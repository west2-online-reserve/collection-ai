# 游戏功能说明 #
## 这个简化版的文字炉石传说包含以下核心功能：

### 基本游戏机制：

* 玩家初始30点生命值

* 每回合增加1点法力水晶（最多10点）

* 每回合抽一张牌

### 卡牌类型：

* 随从牌：有攻击力和生命值，召唤后可以攻击

* 法术牌：有特殊效果（目前简化处理（太难的我不会🥺））

### 战斗系统：

* 随从可以攻击敌方随从或敌方英雄

* 随从召唤的回合不能攻击

* 随从攻击时会互相造成伤害

### 游戏流程：

* 玩家可以打出卡牌、使用随从攻击、使用英雄技能或结束回合

* 当一方生命值降至0或以下时游戏结束
