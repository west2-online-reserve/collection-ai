# 📌 函数应用实战 — 知识点总结

## 例子1：随机验证码

- **核心函数**：`random.choices`（有放回抽样）、`random.sample`（无放回抽样）

- **字符集**：`string.digits`（0–9）、`string.ascii_letters`（大小写字母）

- 函数设计

  ：

  - 使用命名关键字参数 `code_len`，默认值为 4
  - 返回指定长度的随机验证码字符串

⚠️ **要点**：`choices` 的参数 `k` 必须显式写出，且可能出现重复字符。

------

## 例子2：判断素数

- **质数定义**：大于 1，只能被 1 和自身整除

- **优化循环**：只需判断到 `√N` 即可

- 函数签名

  ：

  ```
  def is_prime(num: int) -> bool
  ```

  - `: int` 和 `-> bool` 是**类型标注**，增强可读性，不影响执行

------

## 例子3：最大公约数 (GCD) 与 最小公倍数 (LCM)

- **最大公约数 gcd(x, y)**：辗转相除法
- **最小公倍数 lcm(x, y)**：公式 `x * y // gcd(x, y)`
- **设计原则**：不同功能应拆分为独立函数，避免混杂

------

## 例子4：数据统计

- 统计指标

  ：

  - 均值 (mean)
  - 中位数 (median)
  - 极差 (ptp)
  - 方差 (var)
  - 标准差 (std)
  - 变异系数 (cv = std / mean)

- 中位数计算

  ：

  - n 为奇数 → 取中间值
  - n 为偶数 → 取中间两个数的均值

- 自由度 ddof

  ：

  - 样本方差/标准差 → `ddof=1`（自由度校正）
  - 总体方差/标准差 → `ddof=0`

⚠️ **要点**：Python 标准库 `statistics` 模块已封装常用统计函数。

## 例子5：双色球随机选号

- **红球池**：1–33，随机选 6 个（`random.sample`）

- **蓝球池**：1–16，随机选 1 个（`random.choice`）

- 函数拆分

  ：

  - `choose()` → 生成一组号码
  - `display(balls)` → 格式化输出

- 调用方式

  ：

  ```
  display(choose())
  ```

  - 先生成号码，再传入显示函数

------

## 🎯 总结

- 函数封装的意义

  ：

  - 提高代码复用性
  - 增强可读性与可维护性
  - 降低重复劳动

- 实践原则

  ：

  - 独立功能 → 独立函数
  - 参数设计 → 默认值 + 命名关键字参数
  - 复杂功能 → 函数间相互调用
